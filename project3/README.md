#  Educational-Kids-Game 🎓 📱👦👧

An Educational 🎮Gamified🎮 app for kids,
To teach them various subjects in a fun encourging way !


The user will start in a login page after logging in,
a friendly UI will face the kids they can easliy check their courses and know thier progress and they will be
encourged by the Gamified methods includes:

1- Login Streak
2- XP(Experience Points) after finishing courses
3- Hearts for watching lessons per day (unlimited for paied accounts)
4- Various Challenges
5- Achievements

## 📱 Features


- 🎮 **A Gamified UI** – A gamified UI to encourge the kids to learn and use the app 🛝.
- ✅ **Simplecity** – The UI is easy to use there is not a lot going on so the kids wont be confused using the app.
- 🥇 **Courses progress tracking** – in the courses screen each course has a it's own progress tracking by using crowns so the parents can check how is their kids doing with the courses and also for encourging the kid👨👦👧👩.
- 🖼️ **Profile** – A profile for the kid where they can see their friends progress !, this method will make the kid competitve and willing to learn more, they can also see their own progress and statistics 📈📈.
- ⚔️ **Challenges** – The kids will have a daily challenges page so they can get a reward based on their completed challenge and the will get an achievments too !!!. 
- 📱 **Bottom Navigation Bar** – so the kids can easliy go through the pages without getting confused or over-complicate the app.
