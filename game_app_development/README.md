# Mental Up – Kids Learning App

A colorful and educational mobile application built with Flutter, designed to help children improve their cognitive skills through interactive challenges, games, and rewards.

## Overview

**Mental Up** is an engaging and fun learning app tailored for children. The app includes skill-based sections such as *Logical Reasoning*, *Artistic Thinking*, and *Verbal Skills*, all organized in a child-friendly design with icons, illustrations, and progress tracking.

## Features

- **BottomNavigationBar** for easy navigation between main pages
- Interactive cards representing learning units
- Locked units with visual indicators
- Progress tracking with crown and XP display
- Clean and responsive UI for kids

## App Pages

### 1. **Login Page**
- Fields for Username and Password
- "Forgot Password" and "Create Account" options
- Welcoming design with illustrations
![login page](screenshot\login.png)
### 2. **Home Page**
- Displays three skill categories: Logical, Artistic, and Verbal
- Each category contains a list of units (some locked)
- When a skill is tapped, the user is navigated to its corresponding page
![home page](screenshot\homepage.png)
### 3. **Skill Detail Page (e.g., Verbal Skills)**
- Shows units in a circular icon layout
- Locked units have a lock icon
- Unlock as progress is made
![verbal page](screenshot\verbalpage.png)
### 4. **Profile Page**
- Displays user info (name, image, username)
- Shows statistics like XP, crowns earned, and achievements
- Allows user to invite friends
![profile page](screenshot\profilepage.png)
### 5. **Challenges Page**
- Shows daily or weekly goals
- Displays achievements using icons and short descriptions
![challenges page](screenshot\challengespage.png)
## Technologies Used

- **Flutter** & **Dart**
- Custom widgets for cards and units
- Asset images (PNG) for icons and illustrations
