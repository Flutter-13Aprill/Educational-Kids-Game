## *Educational Kids Game App Development*

 - #*Project description* 

 The Educational Kids Game is a Flutter-based mobile app designed to enhance children's learning through interactive challenges and skill-building exercises. It features secure user authentication, a dashboard displaying various skill categories with progress indicators, and profile management for tracking statistics and friends. The app offers engaging challenges that reward users with achievements and XP, all accessible via a user-friendly bottom navigation bar. With an attractive and responsive design, the app aims to make learning enjoyable and effective for children and their parents. 


> - #*Setup instructions*

1-Clone the repository Through commands line:git clone Your Own path

2-Install dependencies Make sure you have Flutter installed. Then run:flutter pub get

3-Run the app You can launch the app using. Then run: flutter run

Note: You need a working emulator or a connected real device.
